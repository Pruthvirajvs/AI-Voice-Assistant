# Code of Conduct

This project welcomes respectful collaboration.

## Expected Behavior

- Be kind and professional.
- Give constructive feedback.
- Respect different experience levels.
- Keep discussions focused on improving the project.

## Unacceptable Behavior

- Harassment or personal attacks
- Discriminatory language
- Sharing private information without permission
- Spam or intentionally disruptive behavior

Maintainers may remove comments, close issues, or restrict participation when needed to keep the project healthy.
