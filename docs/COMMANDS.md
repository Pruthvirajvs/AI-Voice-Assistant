# Voice Commands

The assistant listens for natural phrases. These are the main supported commands.

## Assistant Basics

| Command | Action |
| --- | --- |
| `jarvis` | Repeat greeting |
| `how are you` | Friendly response |
| `who are you` | Assistant introduction |
| `what's your name` | Speaks the assistant name |
| `change my name` | Updates the user's name |
| `time` | Speaks the current time |
| `exit` or `stop assistant` | Stops the assistant |

## Web Commands

| Command | Action |
| --- | --- |
| `open youtube` | Opens YouTube |
| `open google` | Opens Google |
| `open instagram` | Opens Instagram |
| `open whatsapp` | Opens WhatsApp Web |
| `open twitter` | Opens Twitter/X |
| `search artificial intelligence` | Searches Google |
| `wikipedia` | Asks what to search on Wikipedia |

## Productivity

| Command | Action |
| --- | --- |
| `write a note` | Saves a note |
| `remember that` | Stores a memory |
| `do you remember anything` | Reads the stored memory |
| `take screenshot` | Saves a screenshot |
| `calculator` | Opens Calculator |
| `ms word` | Opens Microsoft Word |
| `powerpoint` | Opens Microsoft PowerPoint |

## System Commands

| Command | Action |
| --- | --- |
| `open c drive` | Opens C drive |
| `open d drive` | Opens D drive |
| `open documents` | Opens Documents folder |
| `open downloads` | Opens Downloads folder |
| `open pictures` | Opens Pictures folder |
| `lock window` | Locks Windows |
| `sleep mode` | Puts the machine to sleep |
| `shutdown system` | Confirms, then shuts down |
| `restart` | Confirms, then restarts |
| `empty recycle bin` | Clears the recycle bin |

## Tools and Fun

| Command | Action |
| --- | --- |
| `weather` | Asks for city and speaks weather |
| `what is ...` | Queries WolframAlpha |
| `who is ...` | Queries WolframAlpha |
| `play music` | Plays a random file from the configured music folder |
| `joke` | Speaks a joke |
| `camera` or `take a photo` | Captures a camera image |
| `download video` | Asks for a YouTube URL and downloads it |
| `add`, `sub`, `mul`, `div` | Performs arithmetic |
